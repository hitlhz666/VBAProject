操作方法：
1、在 Excel 菜单栏中选择 开发工具 > Visual Basic > 模块 插入bas文件；
2、在 VBA 菜单栏中选择 工具 > 引用，勾选 Microsoft Visio xx.x 类型库；
3、运行。
